# sane_cups_bash_editor
Bash tool to configure SANE and CUPS via CLI.

## Quick install:
```bash
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/DemonDED/sane_cups_bash_editor/refs/heads/master/installer.sh)"
```

## Manual install:

1) Download scbe.sh
2) Open terminal and enter - mv scbe.sh /usr/local/sbin/scbe
